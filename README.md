# 🏛 Platform Architecture

```mermaid
flowchart LR

Developer --> GitHub

GitHub --> GitHubActions

GitHubActions --> ArtifactRegistry

ArtifactRegistry --> ArgoCD

ArgoCD --> GKE

GKE --> GatewayAPI

GatewayAPI --> Vote

GatewayAPI --> Result

Vote --> Redis

Result --> PostgreSQL

Prometheus --> Grafana

Kyverno --> GKE

ExternalSecrets --> SecretManager

KEDA --> Worker

Kubecost --> GKE
```
